PoC code for parsing steeplechase logs and submitting to treeherder

steepleherder.ini:

    [Credentials]
    key = <oauth key>
    secret = <oauth secret>

    [Repo]
    project = <project name>
    host = <host name>

    [System]
    autDir = <directory AUT is stored in>
    testsDir = <directory tests are stored in>
    logFile = <name of log generated by the run>
